# 🚗 CarDekho UI Clone

> A frontend UI replication of the CarDekho website built for practice, focusing on responsive design, layout structuring, and component-based development using modern web technologies.

---

## 📌 About the Project

This project is a **practice-based UI clone inspired by CarDekho**, one of India’s leading automotive platforms. The goal was to understand how real-world websites structure their layout, display data, and provide a smooth user experience.

CarDekho is an end-to-end platform that helps users explore cars, compare features, and make informed buying decisions based on price, specifications, and reviews ([Infobip][1]).

This project focuses only on the **frontend UI replication**, not backend functionality.

---

## ✨ Features

* 🔍 Homepage layout similar to CarDekho
* 📱 Fully responsive design (mobile + desktop)
* 🧩 Component-based architecture
* 🎨 Clean UI using modern styling techniques
* 📊 Structured layout for listings and content sections

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/sumayyatabasum/cardekho.git
```

### 2. Navigate to project folder

```bash
cd cardekho
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```

Open 👉 http://localhost:5173
